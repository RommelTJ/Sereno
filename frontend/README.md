# Sereno frontend

React 19 + Vite + TypeScript + Tailwind CSS v4, linted with oxlint and tested
with Vitest. All commands are normally run through Docker Compose from the
repository root — see the [root README](../README.md).

The design tokens (colors, type, radii) live in `src/index.css` as a Tailwind
`@theme` block, sourced from `docs/design/design-handoff.md`.
